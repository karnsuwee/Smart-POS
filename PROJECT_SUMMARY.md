# Smart Shop POS

## What It Does

Smart Shop POS is an AI-powered point-of-sale web application for small food and drink shops. Shop owners can manage their own menus, select dine-in tables or takeaway orders, check out bills, and view dashboard insights. The app uses AI to analyze best-selling menus, peak hours, and peak days, then suggests stock planning strategies to reduce waste and lower ingredient costs.

## Features Built

- User register, login, and logout
- Shop-specific menus, tables, orders, and favorites
- POS workflow: dine-in/takeaway, table selection, cart, checkout
- Favorite menu items saved per shop account
- Dashboard with sales summary, top menus, and hourly sales chart
- AI stock insight from external Gemini API
- Dark mode toggle
